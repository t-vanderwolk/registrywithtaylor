# TMBC Travel System Compatibility Matrix

**Manufacturer-verified stroller ↔ infant car seat compatibility**
Generated for the TaylorMade Baby Co. Travel System Checker. Read-only research — no database, product, adapter, retailer, or compatibility records were modified.

---

## How to read this

Each stroller lists its **manufacturer-supported** infant car seats under three statuses:

- **No Adapter Required** — the seat clicks directly onto the stroller frame.
- **Adapter Included** — compatible, and the needed adapter ships with the stroller.
- **Adapter Sold Separately** — compatible, but the adapter is a separate purchase (exact adapter named).

Rules applied: matched by **model, not brand alone**; only **manufacturer-supported** combinations; **no inferred** compatibility; **no convertible car seats**, **no travel-system bundles as products**, **no accessories**, **no Doona**. Ambiguous or unverified combinations are flagged inline as ⚠ and collected at the end.

> **Scope note:** This matrix is keyed to the brand/model list reconstructed from the catalog import report (the Stroller-table source). Once you paste the exact `brand, model` DB export, model-level rows can be reconciled 1:1. Models present in your DB but not individually verified below are listed under *Missing / needs verification*.

## Car seat family reference

**Universal / "Euro-click" family** — accepted by most premium strollers via a brand-specific adapter:

- **Nuna** PIPA, PIPA Lite, PIPA Lite LX, PIPA RX, PIPA Aire, PIPA Aire RX, PIPA urbn
- **Cybex** Aton (Aton 2 / Q / M / G / B2), Cloud (Cloud Q / G / T)
- **Maxi-Cosi** Mico (Mico 30 / Max / Max Plus / XP / Luxe / Coral XP), Peri 180, Pebble (EU)
- **Clek** Liing, Liingo
- **Britax** Willow / Willow S / Willow SC / Cypress *(only where a brand explicitly lists it)*

**Separate (closed) families** — generally only fit same-brand strollers (own click system):

- **Chicco** KeyFit / KeyFit 30 / KeyFit 35 / Fit2
- **Graco** SnugRide Click Connect / SnugRide SnugFit 35
- **Peg Perego** Primo Viaggio 4-35 (Nido / Lounge)
- **UPPAbaby** Mesa / Mesa V2 / Mesa Max / Aria
- **Evenflo** LiteMax / SafeMax / SecureMax / Shyft (DualRide)
- **Baby Trend** EZ-Lift / Secure-Lift / Flex-Loc / Ally 35
- **Orbit Baby** G3 / G5 infant car seat
- **Nuna** (Nuna strollers are a closed ecosystem — Nuna seats only)
